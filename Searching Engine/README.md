1. 安裝Python虛擬環境套件
$ easy_install virtualenv

2. 建立Python虛擬環境
$ virtualenv [指定虛擬環境的名稱]

3. 啟動虛擬環境
$ cd [指定虛擬環境的名稱]
$ source bin/activate
若要退出則輸入：
$ deactivate

4.在虛擬環境安裝需要的Python套件
$ pip install requests
$ pip install beautifulsoup4

5.使用 a.py
$ python a.py
site:.gov.tw inurl:???

…  

6. 輸出結果
結果會輸出至Output.txt
